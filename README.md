# Guess-My-Number
Guess My Number is a fun and interactive browser-based mini game built with vanilla JavaScript, HTML, and CSS. The objective is simple: guess the secret number between 1 and 20 before your score runs out!

Designed as a beginner-friendly project, this game demonstrates core programming concepts such as:

DOM manipulation

Event handling

Conditional logic

State management
